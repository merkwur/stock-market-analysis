## stock-data-analysis

Simple stock market data analysis tool. 


Add a flags to customize the plots.

```
-p      # Plots the candlestick chart
-er     # Plots the EMA Ribbon
-bb     # Plots the Bollinger Bands
-rs     # Plots the RSI
-md     # Plots the MACD
-pall   # Plots all 
-c      # prepares the contingency table
-test   # test case
```
