## stock-data-analysis

Simple stock market data analysis tool. 


Add new flags to customize the plots.

```
-p      # plots candlestick chart
-er     # plots the EMA Ribbon
-bb     # plots the Bollinger Bands
-rs     # Plots the RSI
-md     # Plots the MACD
-pall   # Plots all 
-c      # prepares the contingency table
```
